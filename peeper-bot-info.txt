CORE FUNCTIONALITY: This is an AutoShardedBot built with discord.py that supports both slash commands (/) and prefix commands (!), featuring a complete economy system with MySQL database backend for persistent data storage, automatic sharding for scalability across multiple servers, comprehensive logging system that tracks all command usage and errors to MySQL, webhook integration for uptime monitoring and status notifications, and a web server running on aiohttp for handling vote webhooks from top.gg and discordbotlist.com with health check endpoints for monitoring services.

ECONOMY SYSTEM: Users start with 100 cash and 50,000 bank capacity, earning money through daily rewards (5,000 every 24 hours), work command (2,000 every 6 hours), fishing/mining/detecting activities that yield collectible items, farming system with 5 crop types (wheat, corn, carrot, pumpkin, rare_flower) that grow over time and generate passive income, PeepMine idle game where users manage mines with upgrades and miners to earn peepcoins, robbery system allowing players to steal 15-40% of another user's cash with success rates affected by items like robbery potions (70% success) and safe crackers (+25% boost), protection potions that make users immune to robberies for 4 hours, trading system via pay/give commands with 5% fee on pay command, selling items back to shop for cash, and a comprehensive inventory system with tools (pickaxe, fishing rod, detector, time turner), boosts (lucky charm +25% rewards for 2 hours, XP boost +50% for 1 hour, stamina potion to reset cooldowns), upgrades (bankcard +100k capacity, vault key +500k capacity), and collectibles (apple, sword, ancient coin, diamond, pearl, crystal ball, sapphire, emerald, ruby, gold bar, peepcoin).

LEVELING & PROGRESSION: Experience points are earned from every command execution with amounts varying by activity (2-8 XP typically), level formula uses squared progression (level² × 100 XP needed), maximum level is 100 where XP gain stops, premium users get +20% XP boost automatically, XP boost items provide additional +50% for 1 hour, level rewards are given at milestones (5, 10, 15, 20, 25, 30, 40, 50, 60, 75, 100) including items like bankcards, vault keys, tools, and collectibles, and the profile command displays detailed stats including level progress bars, earned balance (money from commands vs trades), wealth rank, inventory value rank, farm profit, mine earnings, game statistics, and earned badges.

PREMIUM SYSTEM: Premium membership can be obtained through redeeming codes (weekly/monthly/lifetime), voting rewards, or referral milestones, providing benefits including 30% reduced cooldowns on all commands, +20% XP boost on all activities, +10% farming yield, 20% faster crop growth, 10 farm plots instead of 5, higher prestige cap in PeepMine (7 vs 5), better PeepMine starting stats (2 miners, 0.15 rate vs 1 miner, 0.05 rate), 2M peepcoin storage vs 250k, can prestige mine at level 90 instead of 100, +15% bonus on PeepMine collection, 15% discount on mine upgrades, 150 max miners vs 100, +25% prestige boost per level vs +20%, gold colored embeds throughout the bot, and premium badge displayed on profile.

FARMING SYSTEM: Five crop types with varying costs, growth times, and yields - wheat (1,200 cost, 2 hours, 1,000/hour yield), corn (3,600 cost, 3 hours, 1,800/hour), carrot (6,000 cost, 4 hours, 2,200/hour), pumpkin (12,000 cost, 12 hours, 1,600/hour), rare flower (48,000 cost, 36 hours, 2,000/hour), users can plant crops in multiple plots (5 for regular, 10 for premium), crops grow in real-time even when offline, automatic DM notifications when crops are ready to harvest, harvest command collects profits and tracks total farm earnings, crops command shows all growing plants with progress percentages and time remaining, farm leaderboard ranks users by total profit earned, and premium users get 20% faster growth and 10% higher yields.

PEEPMINE IDLE GAME: Passive income system where users manage a mine that generates peepcoins automatically, mine level upgrades increase mining rate by 0.03/sec (costs level × 8,000 peepcoins, max level 100), hiring miners multiplies production (costs 8,000 × current miners, max 100 regular or 150 premium), prestige system resets mine to level 1 but grants permanent +15% boost per prestige for regular users and +25% for premium (max 5 regular, 7 premium), peepcoins are stored up to capacity (250k regular, 2M premium) and must be collected, collected peepcoins are added to inventory as tradeable items, total lifetime earnings tracked on leaderboard, premium users get 15% collection bonus, 15% upgrade discount, +25% prestige boost, and can prestige at level 90, interactive buttons for collect/upgrade/hire/prestige actions, and real-time earnings calculation based on time passed.

GAMES WITH BETTING: Rock Paper Scissors against bot or players with optional bets, Tic Tac Toe with 3×3 grid and bot AI for solo play, Connect 4 with 6×7 board and strategic bot opponent, Guess the Number (1-100 in 7 attempts), Hangman with 20-word dictionary and 6 lives, Blackjack with full deck simulation, hit/stand/double actions, 3:2 blackjack payout, dealer stands on 17, all games support multiplayer with challenge system requiring opponent confirmation, betting system where winner takes all (loser's bet goes to winner), ties refund both players, 15-second cooldowns between games, game wins tracked separately for each game type, badges awarded at 50 wins per game (Tic Tac Toe God, Column Conqueror, Dealers Bane, RPS Champion, Number Wizard, Word Master), 3-minute timeouts with automatic forfeit, and XP rewards for playing (5 XP per game).

COMMAND FEATURES: 47 total commands all supporting both slash (/) and prefix (!) formats, comprehensive help command with paginated display showing 6 commands per page with emojis, descriptions, and aliases, ping command showing bot latency, profile command with detailed stats, wealth, inventory, farming, mining, game wins, badges, and premium status, balance command showing cash, bank, capacity, usage percentage, level, and XP progress, inventory command with categorized display (collectibles, boosts, tools) sorted by quantity, shop with 4 pages of items, announcement system for bot updates, bug report command with reward information and web form link, badges command showing all badges and progress, and boosts command displaying active effects with timers.
